1️⃣ Proposed Solution & App Overview

SkyCAst is an advanced Android weather application that combines traditional forecasting with modern interactive features.

  • 👥 Interactive Community Features
  • 🎥 Immersive Weather Backgrounds
  • 📱 Advanced Offline Capabilities
  • 🌬️ Real-time Air Quality Monitoring
  • 🚨 Intelligent Weather Alerts

2️⃣ Application Flow & Architecture

🚀 APP LAUNCH
   ├── 🔐 Firebase Authentication
   ├── 📍 Location Services
   ├── 🌍 Weather Data Fetching
   ├── 🎨 UI Presentation
   ├── 👥 Community Features
   ├── 🗺️ Interactive Maps
   └── 🚨 Alert System

3️⃣ Technology Stack

  • ⚡ Java + AndroidX (Core App Development)
  • 🔥 Firebase (Auth, Firestore, Messaging, Storage)
  • 🌐 OpenWeatherMap API (Weather + AQI Data)
  • 🗺️ Google Maps SDK (Radar & Overlays)
  • 🎨 Lottie + Material Design 3 (Modern UI/UX)

4️⃣ Unique Features
